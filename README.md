# Sudoku Solver

## Description

A Python-based Sudoku Solver that automatically solves an incomplete Sudoku puzzle using the Backtracking Algorithm.

## Features

* Solves 9×9 Sudoku puzzles
* Uses Backtracking for efficient solving
* Displays the original puzzle
* Displays the completed solution
* Handles valid Sudoku inputs

## Technologies Used

* Python

## How to Run

1. Clone the repository:
   git clone https://github.com/guru-sud/PRODIGY_SD_04.git

2. Open the project folder.

3. Run:
   py sudoku_solver.py

## Algorithm

The program uses the Backtracking Algorithm:

1. Find an empty cell.
2. Try numbers 1–9.
3. Check if the number is valid.
4. Continue recursively.
5. Backtrack if no valid number exists.
